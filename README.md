# Josh Espanola — Portfolio

This is a React single-page portfolio project scaffolded for Josh Espanola (student multimedia developer). It uses functional React components, Framer Motion for animations, and accessible UI patterns.

Quick start:

```bash
npm install
npm start
```

Build for production:

```bash
npm run build
```
